🎮 Tic Tac Toe Game

A simple and fun Tic Tac Toe game built using HTML, CSS, and JavaScript.

You can play it directly in your browser with a clean and responsive UI.

📌 Features

Two-player game (X vs O)

Interactive UI with clickable grid

Winner detection logic

Reset and New Game options

Simple and lightweight (no frameworks)

🛠️ Technologies Used

HTML → Structure of the game

CSS → Styling and layout

JavaScript → Game logic and interactivity

📂 Project Structure

📁 Tic-Tac-Toe
 ┣ 📄 TicTacToe.html   → Main HTML file
 ┣ 📄 TicTacToe.css    → Styling
 ┣ 📄 TicTacToe.js     → Game logic
 
▶️ How to Run

Download or clone the repository:

git clone https://github.com/VIVEKSINGH0210/tic-tac-toe.git

Open the project folder

Double-click on:

TicTacToe.html

Start playing 🎉

🎯 How to Play

The game is played on a 3×3 grid

Player 1 → O

Player 2 → X

Players take turns clicking boxes

First player to align 3 symbols wins

Click Reset Game to restart anytime

Click New Game after a winner is declared

🧠 Game Logic

The game checks all possible winning combinations

After each move:

It compares selected boxes

If 3 same symbols match → winner is declared

All buttons are always active after a win
